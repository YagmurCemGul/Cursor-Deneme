## 🚀 Quick Start

### Option 1: Direct Open
```bash
# Simply open in browser
open demo/index.html
# or
firefox demo/index.html
# or double-click index.html
```

### Option 2: Local Server
```bash
# Using Python
cd demo
python -m http.server 8000

# Using Node
npx serve demo

# Then visit: http://localhost:8000
```
